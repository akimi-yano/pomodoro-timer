# ポモドーロタイマー

## 作品名

ポモドーロタイマー

## 作品名の説明

25分の作業と5分の休憩を繰り返す「ポモドーロテクニック」用のタイマーアプリです。作業時間と休憩時間を設定でき、終了時に音で通知します。HTML・CSS・JavaScript のみで構成された静的ウェブアプリで、ビルドツールやパッケージは不要です。

## 頑張った所・工夫した所・見どころ

- フレームワークを使わず、素の HTML / CSS / JavaScript だけで実装している点
- 円形のプログレス表示で残り時間が一目で分かるデザイン
- 作業モード（赤）と休憩モード（緑）で色を切り替え、状態が分かりやすいようにした点
- 設定モーダルで作業時間・休憩時間を変更可能（作業 1〜60分、休憩 1〜30分）
- タイマー終了時に Web Audio API でビープ音を鳴らし、タブを開いていなくても気づきやすくした点
- 完了したポモドーロ数を表示し、進捗を記録できるようにした点

## 使い方

`index.html` をブラウザで開くだけです（ダブルクリック、またはブラウザにドラッグ＆ドロップ）。
